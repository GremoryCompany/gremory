# SITE (Front) — Hospedar na Vercel

Este pacote é **apenas o site** (HTML/CSS/JS). Ele chama o BOT/API por HTTP.

## Onde hospedar
- Vercel (Static site)

## Configuração
Edite `config.js`:
- `window.API_BASE` = URL do seu BOT/API (Render/Railway/Fly/VPS)
- `window.API_KEY` = mesma chave que você colocou no BOT (env `API_KEY`)

## Deploy na Vercel
- Crie um projeto na Vercel
- Faça upload desse pacote (ou conecte GitHub)
- Framework: **Other**
- Output: static (padrão)
